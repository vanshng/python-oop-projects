# 🏦 Bank Management System

## 📌 Description

A simple Python Object-Oriented Programming (OOP) project that simulates basic banking operations.

## ✨ Features

- Create a bank account
- Deposit money
- Withdraw money
- Display current balance

## 🛠️ Concepts Used

- Classes
- Objects
- Constructor (`__init__`)
- self
- Methods

## 📂 Project Structure

```
Bank_Management_System/
├── main.py
└── README.md
```

## ▶️ How to Run

```bash
python main.py
```

## 🚀 Future Improvements

- Prevent withdrawing more than the available balance.
- Add transaction history.
- Save account details using file handling.
- Add multiple bank accounts.
