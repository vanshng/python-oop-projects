# 🎓 Student Management System

## 📌 Description

A simple Python OOP project to store and display student information.

## ✨ Features

- Add student details
- Display student details
- Introduce the student

## 🛠️ Concepts Used

- Class
- Object
- Constructor (`__init__`)
- self
- Methods

## ▶️ Output

```text
Name : Vansh
Age : 20
Course : BCA

Hi, I am Vansh and I am studying BCA.
```

## 🚀 How to Run

```bash
python main.py
```
